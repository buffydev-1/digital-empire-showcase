# MAYA CHEN | Concept Portfolio by BUFFY

A high-end, responsive portfolio demonstration focused on sophisticated UI/UX interactions, glassmorphism aesthetics, and fluid motion design. This project serves as a visual and technical showcase of modern front-end capabilities using core web technologies.

## Technical Philosophy

While my production-grade applications are typically architected using **Next.js** to ensure maximum security, server-side rendering (SSR), and scalable infrastructure, this specific project was intentionally built using **Vanilla HTML5, CSS3, and JavaScript**. 

The goal of this repository is to demonstrate:
* High-performance DOM manipulation without heavy framework overhead.
* Advanced CSS architecture including custom properties and complex grid systems.
* Native Intersection Observer API for scroll-driven animations.
* Mathematical interpolation for custom cursor physics and magnetic interactions.

## Key Features

* **Dual-Layer Interactive Cursor**: A custom-engineered cursor system with physics-based following logic.
* **Fluid Motion Engine**: Seamless scroll-reveal animations optimized for 60fps performance.
* **Magnetic Component Logic**: UI elements that respond dynamically to proximity, enhancing user engagement.
* **Glassmorphic UI**: Multi-layered transparency and backdrop-filter effects for a premium aesthetic.
* **Responsive Typography**: Adaptive font scaling using CSS clamp functions for consistent hierarchy across all devices.

## Project Structure

* `index.html`: Semantic structure and SEO-optimized markup (Concept Brand: Maya Chen).
* `style.css`: Modular CSS architecture featuring over 350 lines of custom styling and layout logic.
* `script.js`: The interaction engine handling the custom cursor, scroll observers, and UI state management.
